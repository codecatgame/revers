# Реверс — сайт для завантаження аудіо за посиланням

## Запуск локально

```bash
cd yt-mp3-site
pip install -r requirements.txt
python app.py
```

Потім відкрий у браузері: **http://127.0.0.1:5000**

## Структура проєкту

```
yt-mp3-site/
├── app.py                # Flask-бекенд (виклик yt-dlp, роздача файлів)
├── requirements.txt
├── templates/
│   └── index.html        # головна сторінка
├── static/
│   ├── style.css          # дизайн "касетна дека"
│   └── script.js          # логіка форми, буфер обміну, полиця треків
└── downloads/             # сюди зберігаються готові mp3 (створюється автоматично)
```

## Як це працює

1. Людина вставляє посилання (вручну або кнопкою "📋" — вставка з буфера обміну).
2. Натискає "Записати доріжку" → фронтенд шле `POST /api/download` з посиланням.
3. Бекенд запускає `yt-dlp` у форматі mp3 і зберігає файл у `downloads/`.
4. Готовий файл одразу зʼявляється на "Полиці" знизу з кнопкою завантаження.

## Важливо

- Це локальний інструмент для персонального використання — завантажуй лише те, на що маєш право.
- Для продакшн-деплою (реальний хостинг) знадобиться чергa завдань (напр. Celery/RQ), бо `yt-dlp` виконується синхронно і довгі відео можуть блокувати запит.
